MIX 1010

This is a simulator for Knuth's MIX machine from the book Art of Computer Programming. We use the identifier 1010 because this machine is slightly different from MIX 1009. We encode masks using binary and there is no division or multiplication instructions.

Statements in the assembler language look like (:label) (/mask) (instruction) (relative address) (+index). Here is an example multiplication routine:

# multiplication routine
:start LDA y
JAZ end
DECA 1
STA y
LDA x
ADD s
STA s
JMP start
:end LDA s
HLT

# data
:s 0
:x 5
:y 6

This is alpha software and there are probably bugs. I plan to polish it over the next few months. The source code can be inspected on github.
