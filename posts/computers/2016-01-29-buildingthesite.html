Building the site

This website is built using Hakyll and Clay.

Hakyll is a static website generation tool and clay is a CSS preprocessor. To build the site from source use the following commands:

$ git clone https://github.com/danielbarter/personal_website_code.git
$ cd ./personal_website_code
$ make build
$ ./website watch

For this to work, you need to install stack which is a build system for the haskell platform. If your terminal prints Success, then point your browser to localhost:8000 and you should see the website.
